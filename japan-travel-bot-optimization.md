# 🇯🇵 日本旅遊 LINE Bot 優化完成

針對您的東京+北海道 12 天自助旅行，我已完成以下優化：

## ✨ 新增功能

### 📸 支援更多文件類型
| 類型 | 說明 |
|------|------|
| 🛒 購物收據 | 藥妝店、便利商店、百貨公司等 |
| 🍜 餐廳收據 | 拉麵店、居酒屋、餐廳等 |
| 🚄 交通票券 | 新幹線、JR、地鐵、巴士、機票 |
| 🏨 住宿確認單 | 飯店、旅館、Airbnb 等 |
| 🎫 景點門票 | 迪士尼、美術館、溫泉等 |

### 💬 快速指令
| 指令 | 功能 |
|------|------|
| `!統計` | 查看各類旅費總計 + 日均花費 |
| `!行程` | 時間軸顯示交通行程 |
| `!住宿` | 查看住宿清單 |
| `!景點` | 查看景點紀錄 |
| `!清空` | 清除所有紀錄 |
| `!幫助` | 顯示使用說明 |

### 🤖 智慧問答
直接輸入問題即可，例如：
- 「今天花了多少錢？」
- 「在藥妝店買了什麼？」
- 「明天的新幹線是幾點？」
- 「推薦北海道的美食」

### 🎯 日本特有功能
- **消費稅識別**：自動識別稅込/稅抜金額
- **免稅金額**：記錄免稅商品金額
- **IC卡支付**：識別 Suica/PASMO 支付
- **日文翻譯**：店名、站名保留日文原文 `中文(日文)`
- **新幹線支援**：自動識別車次、座位等級

### 📱 Quick Reply 按鈕
每次回覆都會顯示快捷按鈕，一鍵操作更方便！

## 📊 Google Sheets 資料表
自動建立 4 個工作表：
1. **Receipts** - 購物/餐廳收據
2. **Tickets** - 交通票券
3. **Hotels** - 住宿紀錄
4. **Attractions** - 景點門票

## 🚀 使用建議

### 出發前
1. 測試 Bot 是否正常運作
2. 確認 Google Sheets 已建立

### 旅途中
1. 每次消費後拍照上傳收據
2. 車票、門票也都可以拍照記錄
3. 隨時用 `!統計` 查看花費

### 回國後
1. Google Sheets 完整記錄所有消費
2. 可作為報帳或回憶的依據
